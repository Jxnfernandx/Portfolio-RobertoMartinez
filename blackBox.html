Black Box Theatre

The Black Box Theatre architectural project draws inspiration from Antoni Gaudí’s use of sandbags on a line. The Black Box Theatre takes planes and places them around a cube in a wire matrix with a line. Each plane is placed in a certain way to make the cube the main body and have each connected planes as an outstretched limb. Crossing the line in the matrix around the cube causes the planes to come out in sharp triangular forms.
